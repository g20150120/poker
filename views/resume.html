About This Website

This website is built 100% by myself. I used Node as server, jade as view engine, a domain from GoDaddy, and a vultr VPS (template came from Ceevee ) and is best viewed in Chrome or Safari. All the pictures, logos, contents are purely original.

The logo on the left is a design of mine. Its shape of sharp-edged square means concision and order; its color in light orange expresses the energetic and passionate part of me; 0 and 1 represents the binary world of programing language; A and Z represents me, Alexander Zhao, and, as the first and last of alphabet, also represents the vivid world depicted by human language.

This website is devoted to sharing something about myself with the world. This project is also a trial to develop and deploy a well-organised, good-looking, and fully operational website from scratch. Image or file transmission might be inefficient or even ineffective sometimes, but please have patience.

Project

Cubot

a robot to solve Rubik’s Cubes • 2016-2017

I used C++ to analyse a scrambled cube and then generate a solution. Later I employed Arduino to control steppers through drivers and an external DC power source to rotate each face of the cube accordingly. All the code were self-written and put together referencing algorithms and documentations online. All the structure and components of this robot were also designs of my own. 3D printing was used to produce some of the components.
